# OpenEMR Patient Registration Manual Testing Project

## Project Overview

This project demonstrates end-to-end Manual Testing of the Patient Registration module in OpenEMR, an open-source Electronic Health Record (EHR) system used in the Healthcare domain.

The objective was to perform a complete Software Testing Life Cycle (STLC) process including:

* Requirement Analysis
* Test Planning
* Test Scenario Design
* Test Case Creation
* Test Execution
* Defect Identification
* Test Reporting

---

OpenEMR Demo Application
